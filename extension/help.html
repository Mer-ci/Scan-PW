Scan.pwd web

Coupler avec l'application android Scan PW web va vous permettre de vous connectez facilement à tous vos sites sans avoir à vous souvenir de tous vos mots de passe

Principe de base

Scan PW web va afficher un QR code quand il détectera le besoin d'entré des identifiants, il ne vous restera plus qu'à scanner ce QR code avec l'application mobile Scan PW pour remplir les champs de connections avec vos identifiants. Vous n'avez aucune configuration à faire, il suffit que l'extension soit installée.

Options

Réglages

Différents réglages sont disponibles à partir de la popup de l'extension, pour l'ouvrir cliquez sur l'icône

réglages possibles :

  • désactiver la détection automatique : si vous voulez utiliser l'extension que sur quelques sites vous pouvez désactiver la détection automatique et afficher manuellement le QR Code en cliquant sur l'icône
  • Changement de la taille du QR : vous pouvez à partir de la fenêtre du QR Code changer sa taille (petit, moyen, grand)

menu contextuel

Si les champs de logins ne se remplissent pas automatiquement après scan de votre QR Code, vous pouvez faire un clic droit sur le champ à remplir (Scan.PW web puis insert user/insert password)
